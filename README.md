# App Numerología

![Node.js CI](https://github.com/studentcs502023/AppNumerologia/actions/workflows/node.js.yml/badge.svg)

Sistema de numerología con autenticación y gestión de pagos.

## Tests
Este proyecto utiliza **Mocha** y **Chai** para las pruebas unitarias.
Husky está configurado para ejecutar los tests automáticamente antes de cada commit.
